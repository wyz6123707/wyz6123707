重装步骤及代码摘要

发表于 2017-08-01 | 分类于 Tech

今天手贱把整个智能家居的平台重装了一次，终于还是毁掉了全部的成果……今天一整天都在走弯路，不知道到底哪里出问题了，现在感觉大概是小米插件的问题。因此记录一下，步步为营。

综述

重装的步骤大概是，刷好树莓派系统，然后连显示器设置好ssh。

然后在电脑上ssh进pi，安装samba，设置samba。

然后安装home assistant，并设置开机启动，然后配置。
然后安装mqtt服务，设置好密码。
然后安装homebridge，并设置开机启动。
然后安装homebridge两个插件，并配置好homebridge。

samba

sudo apt-get install samba samba-common-bin
sudo nano sudo /etc/samba/smb.conf
修改配置文件：

sudo smbpasswd -a root
sudo reboot

home assistant

sudo apt-get install python3-pip
sudo pip3 install homeassistant
hass

检查浏览器

设置开机启动

sudo nano /etc/init.d/hass-daemon
粘贴以下内容

sudo chmod +x /etc/init.d/hass-daemon
sudo update-rc.d hass-daemon defaults
sudo service hass-daemon install
注意

配置

配置的部分看自己的造化了，慢慢来。

mqtt

sudo apt-get install libssl-dev libc-ares-dev uuid-dev g++
wget http://mosquitto.org/files/source/mosquitto-1.4.tar.gz
tar zxfv mosquitto-1.4.tar.gz
cd mosquitto*
make
sudo make install
sudo ln -s /usr/local/lib/libmosquitto.so.1 /usr/lib/libmosquitto.so.1
sudo ldconfig
sudo mosquitto_passwd -c /etc/mosquitto/passwd mqtt

homebridge

sudo apt-get install git make
curl -sL https://deb.nodesource.com/setup_7.x | sudo -E bash -
sudo apt-get install -y nodejs
sudo apt-get install libavahi-compat-libdnssd-dev
sudo npm install -g –unsafe-perm homebridge hap-nodejs node-gyp
cd /usr/lib/node_modules/homebridge/
sudo npm install –unsafe-perm bignum
cd /usr/lib/node_modules/hap-nodejs/node_modules/mdns
sudo node-gyp BUILDTYPE=Release rebuild

安装插件

sudo npm install -g homebridge-broadlink-rm
sudo npm install -g homebridge-homeassistant

设置开机启动

sudo apt-get install screen
sudo nano /etc/rc.local
在最底下 exit前面加入这一行
su -c “screen -dmS homebridge homebridge” -s /bin/sh pi

修改以兼容script和automation

# homebridge # homeassistant # homekit
