为自己写的配置单

发表于 2017-05-24 | 分类于 Tech

从上个学期初开始，我就想装台新电脑，画图时深深感觉电脑已经跟不上我的手速了，想玩的游戏也没啥玩的了的。但是苦于钱不够，当时买了机箱和电源就停下来了。现在再重启装机计划时，内存的价格已经翻了一番，原本买的起的东西现在也买不起了。不管怎样，降价减配也是可以接受的。
先上配置单：

这是一套配给AMD新平台Ryzen的PC，华硕主板，贼船3000内存，微星1066显卡，选择了性价比很高的追风者416侧透机箱。

CPU

英特尔常年挤牙膏已经给AMD制造了大量反击的机会，虽然我们清楚地知道AMD离逆袭英特尔还遥不可及，但是这次AMD推出的Ryzen系列CPU显然给了英特尔致命一击。我选择的Ryzen7 1700虽然主频只有3.0，但是可以轻松超频到3.85，换个好点的散热超到4都不是问题。8核心16线程也远远把7700k甩了几条街，在PS中，多核优化还是做得很好的。

主板

我知道华硕主板贵，但其实一开始我没想过选主板的事，一块1700要两千多，CPU主板的套装才2600多（发文时已经降至2575），所以就选了这套华硕B350-PLUS + R7 1700，主板虽然没有感觉很好，但是至少对得起华硕这个牌子，用料不算太差，超频也没问题。

四条内存槽，兼容到64G，两个M.2槽，CPU3+3供电，完全够我用了。

内存

1700和7700k的差距显然可以靠高频内存来压缩一下，我选择贼船LPX3000的8G*2套条，一是因为贼船口碑好，二是因为实际上他妈的这套条才800多块，就当下内存的行情来说，性价比太高了。16G单条的假士顿FURY 16G 2400的条子也要800多，没啥好说的。

但是不得不说的是，Ryzen对高频内存的兼容还不太好，很多人拿着3000或者3200的高频条上不了那么高的频率，降到2933甚至2666的比比皆是。没有高频条，1700那什么和7700k比？这个问题让我纠结了很久，屡屡放弃ryzen平台。

散热器

1700超频的话，散热确实是个问题，我一直以为我原有的那个t400i能派上用场。后来我才想起来，我这块散热器是intel平台专用的，全年买的时候也没想过AMD会半路杀出个程咬金啊。反人类400这种变态的高度也不是什么机箱都hold得住的，没办法，准备出掉吧，1700原配的散热估计也能卖个八九十。
所以散热我选择了t400pro，好看好用又便宜，应该能压得住超频的1700。如果实在压不住的话，出掉上水冷吧。

显卡

一开始觉得1050ti就够了，可是现在觉得1066才够，在这个问题上我从没纠结过1063和1066，3G显存显然很多游戏都吃不消。
老机器上GTX660M的显卡虽然不算太差，但是已经彻彻底底被这个时代抛弃了，别说各种大作了，就几年前普通的游戏我都不敢调高画质玩，因为根本没法玩……
要不要买屌丝最爱的影驰，还是买微星的屌丝版1066我纠结了挺长一阵子，其实到现在我都还没决定，两块显卡感觉没多少不一样，不过影驰的RGB灯效好看。

机箱电源

机箱电源是老早就买好了的追风者416和振华金山冰蝶战斗版450w，不知道是不是所谓的战斗蝶，但是用了一阵子感觉很不错完全没声音，所以准备留着用了。
416这款机箱真的非常满意，钢板很厚实，背部空间非常充足，机箱很大且刚好可以放进我宿舍的机箱位。机箱内部有两个抽屉式的硬盘槽，背板可以挂两块2.5的SSD/HDD，CPU散热兼容反人类400，显卡可以37CM，没啥好说的了……真的又强又骚。机箱还送了个灯条，贴在机箱内部，RGB灯效。

前一周出大型公建的图的时候，各种卡各种慢，mac虽然优化好，但是这种大规模的工作也完全吃不消，老神船毕竟老了，三代i7和8G内存显然也吃不消了。特别的是在用PC的时候，修改字体，渲染一下都要十几二十秒，特别难用。

我用我的老爷机玩极品飞车18时，画质调到最低，还有三五十帧，挺流畅的，但是画质调低的后果就是地图上有一些模型看不清楚，经常撞到草后边的电线杆然后被警察抓走。这游戏里有一片小树林，每次一进那片小树林就卡得不行，剩下十几帧，各种转弯转不过来，然后出了小树林就撞到草后面的电线杆被警察抓走。这是一大痛点。

总之，等端午假期回家下单，返校时大概这堆东西就到了。
